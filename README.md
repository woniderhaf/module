## Используемый стэк
1. React Native - 0.73.1
2. Redux Toolkit - 2.0.1
3. React Navigation - 6.1.9
4. TypeScript
5. Reanimated - 3.6.1
6. Gesture Hundler - 2.14
7. React Hooks
8. Formik - 2.4.5
9. Axios - 1.6.3
